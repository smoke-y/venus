# VENUS
This is a simple solution I found to solve a problem I had while distro-hopping: tracking config files in a centralized repo.

## USAGE
Clone the repo and cargo run. This will create `conf/.config`, where you can add the paths of all the files you would like to track.
run the app again to copy all the required files under `conf/`. You can now use your desired software to track diffs under this
folder.
